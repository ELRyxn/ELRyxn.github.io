Oefening 1

De accordion DOM-tree besteet uit 3 elementen:

  • De 1ste is het Head element.
  • De 2de is het DIV element.
  • Het 3de element is een footer element.

Oefening 2

Als je een gesloten accordion opend wordt deze toegevoegd aan de actieve accordions.
Als er een ander accordion actief is wordt deze gesloten.

Oefening 3

Het HTML document wordt dynamisch aangepast door het gebruik van javascript.
Het gedrag van de pagina wordt zo aangepast, waardoor de broncode en html document verschillen.
